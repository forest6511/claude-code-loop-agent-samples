# 仕様: 頻出メッセージ上位表示（id: top-messages）

- count に --top N オプションを追加する
  例: python3 loglens.py count sample.log --top 3
- message ごとの出現件数を数え、件数の多い順に上位 N 件を
  レベル別集計の後に表示する
- 表示形式は1行1件で「件数 半角スペース メッセージ」
- 件数が同じメッセージは、ログで先に出現した順に並べる
- N がメッセージの種類数より大きいときは、ある分だけ表示する
- --level / --since / --until / .jsonl 入力 / 複数ファイルと併用できる
  （絞り込み後の行を集計対象とする）
- OTHER に分類される行（形式外の行）は集計対象に含めない
- テストで挙動を固定する。既存のテストは変更しない
